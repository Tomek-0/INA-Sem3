## Chinese checkers

Client-server implementation of Chinese checkers game. 

By Manfred Gawlas and Tomasz Piotrowski.

### Applications
System consists of two applications. Client and Server.

To compile and run programs:
```bash
cd server # or cd client

cd build
cmake ..
make
./server # or ./client
```
